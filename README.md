# NotatkaHUB

<img src="https://github.com/1ArtEm-fastC0de1/34-practice/raw/main/docs/images/resources/SE-logo-transparent.png" width="200" align="right" style="float:right;">

Репозиторій з матеріалами для практики з "Технологій розробки програмного забезпечення" (ІІ семестр 3 курсу).

**Проект:** NotatkaHUB - веб-застосунок для створення та управління Markdown-нотатками  
**Команда:** СВАРЩИКИ  
**Статус:** Завершено

## **Опис проекту**

NotatkaHUB - це сучасна веб-система для ведення нотаток у форматі Markdown з автоматичним збереженням, експортом та адаптивним інтерфейсом. Застосунок розроблений як React SPA з підтримкою реєстрації користувачів та персональних акаунтів.

### **Основні можливості**

- ✅ Створення, редагування, перегляд та видалення нотаток
- ✅ Автоматичне збереження під час редагування
- ✅ Підтримка Markdown-форматування з перемиканням режимів редагування/перегляду
- ✅ Експорт окремої нотатки або всіх нотаток у форматі `.md`
- ✅ Адаптивний інтерфейс для мобільних та настільних пристроїв
- ✅ Світла та темна теми
- ✅ Реєстрація та авторизація користувачів
- ✅ Персональне зберігання нотаток у базі даних

## **Технологічний стек**

### Frontend
- **React** - компонентна архітектура UI
- **TypeScript** - типізована мова програмування
- **Vite** - швидка збірка проекту
- **CSS Modules** - модульна стилізація
- **react-markdown** - обробка Markdown
- **Axios** - взаємодія з API

### Backend
- **Node.js** - середовище виконання
- **Express.js** - REST API фреймворк
- **MongoDB** + **Mongoose** - база даних та моделювання
- **JWT** - авторизація та аутентифікація
- **bcrypt** - хешування паролів

### Інструменти та сервіси
- **Git/GitHub** - контроль версій
- **MongoDB Atlas** - хмарна база даних
- **Vercel** - розгортання фронтенду
- **VS Code** - IDE
