# Список питань для інтерв'ю
Команда «*СВАРЩИКИ*»

1. Чи потрібен пошук по нотатках (ключові слова, теги)?
2. Чи має бути можливість сортування/фільтрації (за датою, назвою, тегами)?
3. Чи потрібна функція «закріпити» нотатку нагорі списку?
4. У якому форматі експортувати нотатки: тільки .md, чи ще .txt/.pdf?
5. Чи потрібна можливість імпортувати нотатки назад у застосунок?
6. Чи має бути пакетний експорт (всі нотатки одним файлом) чи окремо кожна?
7. Чи потрібна підтримка drag-and-drop для нотаток?
8. Як має виглядати перемикання між режимами редагування та перегляду?
9. Чи потрібна автозбереження під час редагування?
10. Чи достатньо двох тем (світла/темна), чи потрібні додаткові кольорові схеми?
11. Чи має користувач сам налаштовувати шрифти, розмір тексту?
