# Список питань для інтерв'ю
Команда «*СВАРЩИКИ*»

1. Опишіть бізнес-проблему, яку має вирішувати наш застосунок
2. Форматування тексту. Як по вашому має це виглядати? Як в Microsoft Word чи документах GitHub?
3. Який функціонал додатка повинен бути і які вікна потрібні? Наприклад: вікно привітання, вікно про розробників тощо.
4. Предложение сделать сервер для хранения данных и CRUD операций, что-бы можно было заходить в аккаунт и редактировать заметки с любого устройства и для надёжности хранения данных
5. Пропозиція створити сервер для зберігання даних і CRUD-операцій, щоб можна було заходити в обліковий запис і редагувати нотатки з будь-якого пристрою, а також для надійності зберігання даних.
6. Куди зберігати нотатки що бул написані в режимі Гість? Чи правильно ми розуміємо, вони будуть зберігатися у localstorage, та можливості користувача у цьому режимі будуть обмежені бо не буде реєстрації у сервісі?
7. Що ви маєте на увазі під можливістю створювати гілки? Наприклад, зробити можливість називати їх чи зберігати ці версії, як в системі контролю Git або створювати дерева думок за допомогою структури даних дерева.
8. Можливість малювати поверх тексту. Чи треба це залишити?
